# Vehicle Registration
<p align="center" color="red">Vehicle registration system to replace recorder deeds</p>

## Installation

Use the package manager NPM, so you need to install [Nodejs](https://nodejs.org/en/download/)

```bash
npm install
```

## Usage

```bash
npm run start
```
